# Megaphone

Megaphone plans and shapes campaign messages from a local source brief and
example corpus.

Use it when you need to turn source material into channel-specific briefs,
drafts, package exports, and measurement notes.

Typical workspace contents:

- source brief and positioning notes,
- example corpus with post metadata,
- campaign calendar inputs,
- generated post-package outputs,
- measurement signals.

Build your own corpus with:

- `docs/megaphone-corpus-building.md`
- `docs/public-quickstart.md`
- `docs/private-workspaces.md`
- `docs/troubleshooting-public-workspaces.md`

Disabling Megaphone in Workshop only hides the tool from the picker. It does not
delete corpora, package outputs, local examples, or credential state.
